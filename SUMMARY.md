# Project Summary

Stack V1 is a tool designed to measure the durability of code contributions. 

Key features include:
* Real time tracking of code analysis jobs.
* Analysis of code survival over a ninety day window.
* Automated filtering of non code files.
* Identification of developers using GitHub IDs and multi repo analysis.
* Recruiter flow with location and language scouting via GitHub Search API.
* Real time status updates via Supabase and a specialized developer boot script.

The project uses Next.js for the interface and a Node.js worker for the backend logic.
